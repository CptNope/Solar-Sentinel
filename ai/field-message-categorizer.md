# SolarSentinel - AI Field Message Categorization

## Purpose

Use Ollama AI to interpret incoming field messages from LoRa and HAM sources.

## Behavior

- Categorize Messages:
  - Waypoints
  - Solar Status
  - System Warnings
  - Node Presence
- Generate Summaries:
  - Daily Ops Briefs
  - Field Mission Logs
- Enable RAG Queries:
  - "Show me the last solar warnings."
  - "List new waypoints added in last 24 hours."
